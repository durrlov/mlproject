End-to-End Machine Learning Project

This web app uses a machine learning model trained on student performance data to predict a student's math score based on demographic and academic features.

Multiple regression models such as Linear Regression, Ridge, Lasso, Decision Trees, Random Forest, AdaBoost, CatBoost, etc. were evaluated.
Lasso Regression was selected as the final model based on its performance (R² ≈ 88.06%) on test data.
